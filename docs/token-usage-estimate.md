# Token Usage Estimate

This document provides a rough estimate of token usage during typical workflows.

## Typical task: graduation poster generation

### Stage 1: requirement analysis
- ~300–600 tokens

### Stage 2: prompt drafting
- ~500–1200 tokens

### Stage 3: iteration (2–5 rounds)
- ~1000–3000 tokens

### Stage 4: refinement and final output
- ~300–800 tokens

## Total per task

Approximate range:
- 2000 – 5000 tokens per complete workflow

## Notes

- Complex identity-preserving tasks require more iterations.
- Visual clarity and prompt precision reduce total token usage over time.
- Structured prompts help maintain consistency and reduce rework.

This estimate is based on repeated usage in creative design scenarios such as poster generation and portrait-based visual concepts.
