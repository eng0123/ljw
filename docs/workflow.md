# Workflow

This document describes the practical workflow used in this repository for AI-assisted visual creation.

## Step 1: Requirement definition

Collect structured input:
- portrait reference (optional but recommended)
- theme (graduation, growth, memory, etc.)
- context (school, major, story)
- preferred style (cinematic, watercolor, minimal, etc.)
- output format (poster, avatar, etc.)

## Step 2: Concept design

- Define a core visual idea (e.g., silhouette double exposure)
- Select limited symbolic elements (3–6 items)
- Decide hierarchy (main vs supporting elements)

## Step 3: Prompt construction

Organize prompt into sections:
- composition
- subject and identity constraints
- scene elements
- style and lighting
- color palette
- typography (if needed)
- negative constraints

## Step 4: Generation and review

Evaluate output based on:
- likeness
- composition clarity
- thematic relevance
- visual clutter

## Step 5: Iteration

- adjust only necessary parts
- preserve working components
- refine prompt progressively

## Step 6: Final output

Deliver:
- final prompt
- description
- optional design notes

This workflow is designed for repeatable use and can be applied across different creative tasks.
