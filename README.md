# AI Poster Agent Skill

A reusable AI workflow for cinematic graduation posters, portrait-based visual storytelling, and structured prompt optimization.

This repository documents a repeatable workflow that turns user-provided portraits, themes, school information, and design preferences into visual concepts and generation prompts.

## What this project does

- Builds structured prompts for graduation posters and commemorative portrait artworks.
- Prioritizes portrait likeness by preserving original facial structure and reference details.
- Produces cinematic double-exposure poster concepts with controlled composition, typography, color, and symbolism.
- Supports iterative refinement around likeness, layout, clutter, and theme adjustment.
- Provides reusable examples for social media avatars, graduation posters, and narrative visual design.

## Typical use cases

1. Graduation poster generation
2. Portrait-based commemorative artwork
3. Social media avatar concept design
4. Prompt engineering for image generation
5. Multi-round creative refinement

## Example workflow

1. Collect user inputs: portrait, school, major, theme, preferred style, and output format.
2. Analyze identity-sensitive details such as face angle, hairstyle, clothing, and expression.
3. Generate a structured prompt with sections for composition, subject, visual symbols, style, color, typography, and constraints.
4. Iterate based on user feedback, especially around likeness, layout, clutter, and theme relevance.
5. Produce final prompt or concept brief for image generation.

## Notes

This project focuses on workflow design, prompt structure, and repeatable AI-assisted creative production. It does not include private user images or third-party assets.
